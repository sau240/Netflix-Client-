# ** Netflix Clone **
A fullstack Netflix Clone using React, Firebase, TMDB API, Redux Toolkit, Node.js, and MongoDB.

## **Features**
- Firebase Signup/Login
- TMDB Movie API Integration
- React Router Navigation
- Save/Delete Movies to MongoDB
- Redux Toolkit for state
- Responsive UI

## **Tech Stack**
- React, Vite/CRA, Redux
- Firebase Authentication
- TMDB API
- Node.js, Express, MongoDB
- MongoDB Atlas (cloud)
- Vercel (frontend), Render (backend)
